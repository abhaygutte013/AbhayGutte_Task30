# React Router Assignment

## About Project

This project is made using React and React Router DOM. The main purpose of this assignment is to learn how routing works in React application. User can move between different pages without refreshing the browser. by using react roter dom i realized how websites works with difffernt diffrent pages on which we go to another tab bon a single tab also.

There are four pages in this project:
- Home
- Dashboard
- Login
- Signup
Each page is connected using React Router.

3.Then open browser and go to:http://localhost:5173
